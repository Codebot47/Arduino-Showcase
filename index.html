Stoplight Circuit

This circuit uses one red, yellow, and green LED, three 220 ohm resistors, and an Arduino Uno R4. The circuit works by supplying and taking away current from the three LEDs. The code first makes the green LED turn on, waits for 2 seconds, then makes the yellow LED turn on, waits for 2 seconds, then finally makes the red LED turn on before it restarts the cycle. I also updated the code I used in the video shown above so that the LEDs fade in and out. I used for loops to do so, with each iteration of the loop increasing the brightness of the LED slightly.
